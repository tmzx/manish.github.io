Data Cleaning Project
in SQL

This project focuses on cleaning and transforming raw data from the world_layoffs database to ensure accuracy, consistency, and usability for analysis and visualization. The process includes identifying and removing duplicates, standardizing data fields, handling null or blank values, and correcting formatting issues such as trimming text and converting data types. The final result is a well-structured dataset ready for meaningful insights and decision-making. Key steps involve:
Removing duplicate records using a combination of `ROW_NUMBER()` and partitioning.
Standardizing industry names and country fields for uniformity.
Transforming date fields from text to proper date formats.
Ensuring null values are addressed logically to maintain data integrity.
 This meticulous approach prepares the data for seamless integration into analytics workflows.
